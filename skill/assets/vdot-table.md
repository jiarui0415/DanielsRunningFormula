## How to use
1. Find runner's current VDOT using Table A below (race performance → VDOT)
2. Use Table B to get all training paces for that VDOT

## Interpolation

For VDOT values between rows, interpolate linearly. For example, VDOT 53:
- T pace ≈ midpoint between VDOT 52 (5:49/mi) and VDOT 54 (5:37/mi) = ~5:43/mi
