# Phase Compression Table (Jack Daniels Figure 10.2)

Source: Jack Daniels' Running Formula (4th ed., 2022), Chapter 10

## Jack's 4-Phase System

Phases progress in this order — **non-negotiable**:
- **Phase I (Base/FIP):** E runs only, build mileage, add strides from week 3+, strength circuit
- **Phase II (IQ — Initial Quality):** Add R (repetition) workouts
- **Phase III (TQ — Transition Quality):** Add I (interval) workouts — most demanding phase
- **Phase IV (FQ — Final Quality):** Shift emphasis to T (threshold), race tune-ups

Full program = 24 weeks (6 weeks each). When fewer weeks are available, compress using the table below.

**Key rule:** Phase III is always the first to be dropped when compressing. Phase IV has priority over Phase II because peak racing fitness (T work + tune-up races) must not be cut.

**Notes:**
- Minimum 3 weeks per phase (below that, the physiological adaptation is too short to be meaningful)
- For 9 weeks: Phase III is skipped entirely — go Phase I → II → IV
- For 6 weeks: Only Phase I and IV — runner gets base mileage then races on it
- For 3 weeks: Only Phase I — build/maintain, then race

---

## Phase entry point (skip phases already completed)

If the runner already has recent training, enter at the appropriate phase rather than starting from Phase I:

| Recent training history | Enter at |
|------------------------|---------|
| Injury / illness / <4 weeks running | Phase I |
| 4–6 weeks E runs only | Phase II |
| 6+ weeks with R workouts | Phase III |
| 6+ weeks with R + I workouts | Phase IV |

When entering mid-program, count remaining phases from entry point, not from Phase I. Example: runner enters Phase III with 12 weeks to race → allocate as Phase III: 6 weeks, Phase IV: 6 weeks.

---

## Race-specific plan lengths

Some race-specific chapters (ch.13, ch.16) have their own fixed plan lengths:
- **5K/10K (ch.13):** 4 phases × 6 weeks = 24 weeks ideal; compress using table above
- **Marathon 2Q (ch.16):** 18-week program (starts at ~Phase III equivalent); don't try to compress below 12 weeks
- **Marathon novice (ch.16):** 18-week minimum; beginners need the full base-building time
